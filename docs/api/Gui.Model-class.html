Class Model

source code

object --+
         |
        Model

The Model keeps track of all of the data that the GUI needs.

This includes the actual behaviour that the GUI is working on, as well as lists that combo boxes will need, and other miscellaneous pieces of data that are used more than once in different areas of the GUI.

UpdateActualVerbNames(self)

source code 

Updates the actual verb names list (actual_verb_names) that combo boxes use.

This should be called any time a change is made to the actual_verbs list.

UpdateVerbNames(self)

source code 

Updates the verb names list that combo boxes use.

This should be called any time the contextual name of a verb in the behaviour object is changed, or a verb is added or removed.

UpdateNWVarNames(self)

source code 

Updates the four lists concerning NWVariable names that combo boxes use.
