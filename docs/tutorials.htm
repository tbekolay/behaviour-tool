Tutorials

If you just want to get started making or editing behaviours, then follow one of the tutorials listed below. If there are any tutorials that you think should be created, please email me!

Creating a New Behaviour

  • If Behaviour Tool is not running
    • Open the behaviour tool.
  • If Behaviour Tool is running
    • Start a new behaviour by clicking on File -> New Behaviour or pressing Ctrl+N

  1. Add the necessary actors and variables first, so that you can use them as arguments to verbs.
  2. Add the necessary verbs.
  3. Connect the verbs by defining which verbs follow each verb. To ensure the verbs relate as you expect them, return to the main behaviour tab and observe the generated graph.
  4. When the objects have been manipulated to your satisfaction, save the behaviour to a directory of your choice; two files (a b_<behaviour name>.nss file and a z_b_<behaviour name>.nss file) will be created and saved in that directory.
  5. Add the behaviour to a module. Open the module in the Aurora Toolset to manually edit sections of the generated script files as necessary.

Editing an Existing Behaviour

  1. Click on File -> Open Behaviour... or press Ctrl+O to open the Open dialog box.

  2. Find the b_<behaviour name>.nss file for the behaviour you want to open, and double click on it.

Re-ordering Verbs

While order does not usually matter, it should be noted that the first verb in the behaviour object will be the first verb executed when a behaviour is started.

  1. Save the behaviour (File -> Save Behaviour, or Ctrl+S), as if something goes wrong during this process, you could lose some of your work.
  2. Delete the verbs that you would like to reorder by click the red X in the top right corner of those verbs' tabs.
  3. Return to the first tab (the behaviour tab).
  4. In the top right hand corner, there's a trash icon. Clicking on it brings up a list of the verbs you've deleted.
  5. Click on a verb's name to restore it. The verb is added to the end of the current list of verbs, so you can restore the verbs in the order you want them in.

Adding a Behaviour to a Module

  1. Using modpacker
    1. Unpack all of a module's files to a certain directory.
    2. Create or open a behaviour in Behaviour Tool.
    3. Save the behaviour in the directory that contains the module's files (File -> Save Behaviour In..., or Alt+S).
    4. Pack the directory into a module.
  2. Using the Aurora toolset
    1. Open the module you want to add your behaviour to.
    2. Create or open a behaviour in Behaviour Tool.
    3. Save the behaviour in <NWN directory>\modules\temp0 (File -> Save Behaviour In..., or Alt+S).
    4. Make some change to the open module (easiest way is to just build it).
    5. Save the module (this packs all of the files in <NWN directory>\modules\temp0 into the .mod file).
    6. Re-open the module.
