Introduction

Behaviours provide a way to quickly author reusable sequences of actions for use in the PaSSAGE framework, or any Neverwinter Nights module. The Behaviour Tool is a tool for the creation and management of behaviours. While the tool is graphical and usable without knowing NWScript, it was not designed as a replacement for the Aurora toolset, so it is expected that users of this program have a working knowledge of NWScript and will be able to use the output generated by the program.

The Behaviour Tool enables storybuilders to author behaviours by defining objects such as actors, using those objects in making verbs, then defining how the verbs interact. The tool also allows you to modify existing behaviours; one can add verbs, modify relationships, and so on. However, it should be noted that changes one makes to a behaviour after it has been generated will not be remembered when one opens the behaviour again in the tool, with the exception of changes made to the opening comment block. For this reason, authors should make note of what sections of a behaviour script file they change after it is generated in case they wish to use the tool to make other changes later on.

Getting Started

The first step to using the tool is to understand behaviours; what they are, why they are used, and how to use them.

Once you understand behaviours, you can jump in to the tool by following one of the tutorials.

For more detailed information about any part of the tool, look in the interface details.
